# RWA Token Explorer

Дашборд, который показывает текущее состояние рынка токенизированных активов.

## Что делает

- Парсит данные с rwa.xyz API / The Graph
- Показывает топ токенизированных фондов (BUIDL, ONDO, etc.)
- График роста TVL по времени
- Сравнение yield vs традиционные инструменты

## Стек

Next.js + The Graph SDK + Recharts + Tailwind

## Контекст: что такое токенизация активов

Токенизация — это создание цифрового представления реального актива (акции, облигации, недвижимость, золото) в виде смарт-контракта на блокчейне.

```
Реальный актив → Custodian (хранитель) → Smart Contract (ERC-20/ERC-1400) → Token на блокчейне
```

**Как физически работает:**

1. **Custodian** (банк/депозитарий) берёт актив под управление и выдаёт подтверждение
2. **Issuer** деплоит смарт-контракт, например ERC-1400 (стандарт для security tokens с KYC)
3. **Oracle** (Chainlink) связывает off-chain данные (цена акции) с on-chain токеном
4. **Transfer Agent** контролирует, кто может владеть токеном (регуляторные ограничения)

**Стек экосистемы:**

- Solidity / Ethereum / Base / Polygon
- Chainlink CCIP (cross-chain) + Price Feeds
- ERC-1400, ERC-3643 (T-REX protocol — стандарт для RWA)
- IPFS для хранения legal документов
- The Graph для индексации событий

**Референс:** BlackRock BUIDL fund — токенизированный фонд денежного рынка на Ethereum, $1 млрд TVL.
